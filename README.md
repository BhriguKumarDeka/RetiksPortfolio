## 🎯 What is RetiX?

**RetiX** is a modern, bento-grid portfolio — handcrafted with precision animations, dual-theme architecture, and a design philosophy that treats every pixel as intentional. It's not a template. It's a statement.

<br/>

## ✨ Feature Highlights

<table>
<tr>
<td width="50%">

### 🌗 Dual-Theme Engine
Seamless dark/light mode with CSS custom properties. Entire palette inverts through a single `data-theme` attribute — zero flicker, instant switch.

</td>
<td width="50%">

### 🧊 Bento Grid Layout
A 4-column CSS Grid system with cards that span, merge, and breathe — creating a magazine-style composition that breaks monotony.

</td>
</tr>
<tr>
<td width="50%">

### 🎭 Framer Motion Choreography
Staggered entrance animations with `containerVariants` and `itemVariants` create a cinematic reveal. Every card arrives with intention.

</td>
<td width="50%">

### 📱 Responsive DNA
Three breakpoints (1024px → 640px → mobile) gracefully collapse the bento grid from 4 → 2 → 1 columns, maintaining visual hierarchy.

</td>
</tr>
<tr>
<td width="50%">

### 🖋️ Typography System
Dual-font strategy: **Outfit** for headings (geometric, modern) and **Inter** for body (clarity, readability) — loaded via Google Fonts.

</td>
<td width="50%">

### 🎨 Micro-Interactions
Hover states that reward curiosity — profile photos de-grayscale, cards levitate, social icons spin, and plus indicators rotate into view.

</td>
</tr>
</table>

## 🛠️ Tech Stack Deep Dive

| Layer | Technology | Why |
|-------|-----------|-----|
| **UI Library** | React 19 | Latest concurrent features, streamlined rendering |
| **Build Tool** | Vite 7 | Sub-second HMR, native ES modules |
| **Animations** | Framer Motion 12 | Declarative, physics-based motion |
| **Icons** | Lucide React | Tree-shakeable, consistent icon system |
| **Styling** | Vanilla CSS | Full control, CSS custom properties for theming |
| **Fonts** | Google Fonts | Inter + Outfit — clarity meets personality |

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Retik29/RetiX.git

# Navigate into the project
cd RetiX

# Install dependencies
npm install

# Launch development server
npm run dev
```

The app will be live at `http://localhost:5173` ⚡

## 🎨 Design Philosophy

```
╭─────────────────────────────────────────────────╮
│                                                 │
│   "Good design is obvious.                      │
│    Great design is transparent."                 │
│                        — Joe Sparano            │
│                                                 │
╰─────────────────────────────────────────────────╯
```

RetiX follows three core principles:

1. **Restraint over Excess** — A monochromatic palette with purpose. Color is earned, not sprayed.
2. **Motion with Meaning** — Every animation serves UX. Nothing moves just to move.
3. **Grid as Grammar** — The bento layout isn't decoration — it's information architecture.
